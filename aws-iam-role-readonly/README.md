# AWS IAM Role readonly

This will create a readonly role, assumeable from `source_account_id`.

## Example

```hcl
module "role" {
  source = "github.com/chanzuckerberg/cztack/aws-iam-role-readonly?ref=v0.11.0"

  source_account_id = "â€¦"
}

output "role_name" {
  value = "${module.role.role_name}"
}
```
