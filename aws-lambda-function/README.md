# AWS Lambda Function

## Example

```hcl
module lambda {
  source = "github.com/chanzuckerberg/cztack/aws-lambda-function?ref=v0.14.0"

  filename         = "file.zip"
  handler          = "handle"
  source_code_hash = base64(hash("file.zip"))
  runtime          = "python3.6"
  timeout          = 10

  project = var.project
  env     = var.env
  service = var.service
  owner   = var.owner
}
```
